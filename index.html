Higgs.IO Documentation - Modern Networking on the JVM Do better.

What?

Higgs is a group of projects built on top of Netty. These are all currently HTTP related but there is a plan to provide a binary protocol for RPC.

Why

I'm not a fan of JSP, servlet containers or Apache's HTTP commons. I like Dropwizard but it's too tightly coupled to Jetty's life-cycle. I also like Jersey. That said, you can see my dilemma. So Higgs takes the best of Dropwizard and Jersey to provide a framework to build REST web services on top of Netty.

What's included?

  1. Core
  2. HTTP client
  3. HTTP server
  4. WebSocket client
  5. WebSocket server
  6. Event Dispatch library
